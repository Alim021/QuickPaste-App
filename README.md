## Project Overview
QuickPaste is a simple and efficient paste sharing web application that allows users to create, share, and manage text snippets online. It features a React frontend and a Node.js backend connected to a MongoDB database.

## Tech Stack
- **Frontend:** React.js
- **Backend:** Node.js, Express.js
- **Database:** MongoDB Atlas
- **Deployment:** Vercel (Frontend), Render (Backend)

## Features
- Create and save text pastes
- Share pastes via unique URLs
- Auto-expiration of pastes (optional)
- Responsive UI for desktop and mobile
